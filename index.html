Setup guides

Install on your home screen
Option A — hosted once, best experience.
  1. Put this single file on any free static host (GitHub Pages, Netlify, Cloudflare Pages). Only the app code is hosted — your expenses never leave this phone.
  2. Open that link in Safari.
  3. Tap Share → Add to Home Screen.
  4. Open it once from the icon — after that it works fully offline.
Option B — no hosting at all.
  1. Save this file in the Files app and tap it to open.
  2. It runs as a viewer. iOS may not keep data between opens this way, so use Save backup after adding expenses and Restore when you return.
Auto-capture bank SMS (Shortcut)
  1. Open the Shortcuts app → Automation tab → New Automation.
  2. Choose Message. Set Message Contains: debited (or your bank's sender ID). Select Run Immediately.
  3. Add action Text → type ===, press return, then insert the Shortcut Input variable.
  4. Add action Append to Text File → File path: queue.txt in Shortcuts folder → Text: the Text block above.
  5. Optionally add Copy to Clipboard (Shortcut Input) + Show Notification ("Expense captured").
  6. Repeat step 2's automation for other keywords (spent, purchase) or senders if needed.
Then in this app: tap Capture from SMS → Import queue file and pick queue.txt. Already-imported messages are skipped automatically, so you never get duplicates.
